# Machine Learning Explorations

A collection of Jupyter notebooks exploring machine learning concepts through mathematical foundations and hands-on implementation. Each notebook contains self-contained explorations of specific topics with from-scratch implementations, visualizations, and detailed explanations.

## Contents

This repository contains various Jupyter notebooks covering topics in:
- **Regression Models**: Linear, Ridge, Lasso, and advanced regression techniques
- **Classification Models**: Logistic regression, SVM, Naive Bayes, Decision Trees, and more
- **Linear Models**: In-depth analysis of linear approaches for supervised learning
- **Neural Networks**: Multilayer perceptrons, activation functions, backpropagation
- **Mathematical Details**: Theoretical foundations, derivations, and optimization theory

### Tech Stack
- **Python 3**
- **NumPy**: Numerical computations
- **Matplotlib**: Data visualization
- **Scikit-learn**: Model implementations and metrics
- **Jupyter Notebook / VS Code**: Interactive development environment

## Key Features
 **From-scratch implementations** of core ML algorithms  
 **Rich visualizations** of model behavior and convergence  
 **Mathematical rigor** with detailed explanations  
 **Practical insights** into hyperparameter tuning and model selection  
 **Hands-on examples** with real data demonstrations  

These explorations are motivated by machine learning coursework, combining theoretical understanding with practical implementation to develop intuition for how algorithms work under the hood.
